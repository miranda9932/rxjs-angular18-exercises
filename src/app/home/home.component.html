Proyecto de ejercicios RxJS para Angular 18

Esta app está pensada para practicar desde lo más básico hasta patrones habituales en proyectos Angular: operadores, Subjects, AsyncPipe, HttpClient, cache con shareReplay, triggers con Subject, HTTP encadenado y filtrado reactivo.

Cómo trabajar

  1. Abre el proyecto en Visual Studio Code.
  2. Ejecuta npm install.
  3. Ejecuta npm start.
  4. Abre http://localhost:4200.
  5. Ve ejercicio por ejercicio y revisa los archivos dentro de src/app/exercises.

La app usa un interceptor mock para simular una API real. Eso significa que puedes practicar HttpClient sin levantar backend.
